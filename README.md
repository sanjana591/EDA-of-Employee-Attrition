# 👩‍💼 Employee Attrition Analysis (EDA)

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on employee data to understand the key factors contributing to employee attrition. The goal is to identify patterns and trends that can help organizations improve employee retention and reduce turnover.

---

## 🎯 Objective
- Analyze employee data to understand attrition trends  
- Identify key factors influencing employee turnover  
- Provide actionable insights for HR decision-making  
- Support future predictive modeling  

---

## 📊 Dataset Description
The dataset includes employee-related information such as:
- Age, Gender, Education  
- Job Role, Department  
- Monthly Income, Salary Hike  
- Years at Company, Experience  
- Work-Life Balance  
- Job Satisfaction  
- Attrition (Target Variable: Yes/No)  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 🔍 EDA Workflow
1. Data Loading  
2. Data Cleaning  
   - Handling missing values  
   - Removing duplicates  
3. Data Exploration  
4. Univariate Analysis  
5. Bivariate Analysis  
6. Outlier Detection  
7. Correlation Analysis  

---

## 📈 Key Insights
- Employees with **low job satisfaction** are more likely to leave  
- **Overtime** is strongly associated with higher attrition  
- Employees with **lower salary** tend to churn more  
- Attrition is higher among employees with **short tenure**  
- Poor **work-life balance** increases the likelihood of attrition
